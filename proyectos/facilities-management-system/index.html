Proyectos¿Quién soy?

Gestión de Facilities

Full‑stack (lead)

Contexto

Registro y gestión de incidencias (tickets), integración con HR para autocompletado de empleados, fotos adjuntas, estados y trazabilidad.

Arquitectura

Frontend Next App Router + API routes; ORM Prisma a SQL Server; autenticación y autorización por roles.

Resultados

  • Implementación completa de formulario público de incidencias con validaciones y preview de imágenes
  • Integración con sistema HR para autocompletar y validar empleados
  • Generación de identificadores de ticket consistentes y trazables
  • Base para panel administrativo y tablero Kanban (siguientes fases)
